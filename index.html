JavaScript tworzenie klas

Opis zadania

Utwórz klasę StringBuilder(baseString) i dodaj do niej następujące funkcje:

  • Wartość domyślna dla parametru baseString to pusty wiersz.
  • Egzemplarz będzie miał właściwość value, w którą wpisuje się wartość parametru baseString.
  • Metoda append(str) - dostaje parametr str (wiersza) i dodaje ją na koniec właściwości value.
  • Metoda prepend(str) - dostaje parametr str (wiersza) i dodaje ją na początek właściwości value.
  • Metoda pad(str) - dostaje parametr str (wiersza) i dodaje ją na początek i na koniec właściwości value.

Ten kod powinien być wykonany bezbłędnie.

Wynik kodu:

Kod utworzy nową instancję klasy StringBuilder, na której wykonane zostaną operacje dopisania, poprzedzenia i okalania łańcuchów. Ostateczny wynik to: "=^.^=".

Rozwiązanie


const consoleLogElement = document.querySelector(".console-log");

class StringBuilder {
  constructor(baseString = "") {
    this.value = baseString;
  }

  append(str) {
    this.value += str;
    return this;
  }

  prepend(str) {
    this.value = str + this.value;
    return this;
  }

  pad(str) {
    this.value = str + this.value + str;
    return this;
  }
}

const builder = new StringBuilder(".");
builder.append("^").prepend("^").pad("=");

console.log(builder.value);
consoleLogElement.innerHTML = ` ${builder.value}`;

Opis

Klasy JavaScript

Klasy JavaScript, które zostały wprowadzone w 2015 roku wraz z wersją ES6, reprezentują nowoczesny sposób tworzenia obiektów. Jest to tzw. "syntactic sugar", co oznacza, że oferuje bardziej przejrzystą i intuicyjną składnię dla tworzenia obiektów, w porównaniu do tradycyjnych metod, które korzystały z konstruktorów o mniej przyjaznej składni.

Konstruktor

Na wstępie stworzono 'konstruktor', który jest funkcją aktywowaną podczas kreowania nowego obiektu. Funkcja ta definiuje atrybuty obiektu, które są dostępne dla każdej instancji klasy. W tym konkretnym przypadku, mamy do czynienia z atrybutem 'value', który przechowuje wartość parametru 'baseString'.

Metody

Metody klasy są zdefiniowane wewnątrz definicji klasy, ale poza konstruktorem, co umożliwia ich dostępność dla każdego egzemplarza tej klasy. W obrębie metod, możemy odwoływać się do atrybutów obiektu za pomocą słowa kluczowego 'this', które wskazuje na specyficzną instancję utworzoną przez klasę. Ważne jest, aby metoda zwracała wartość na koniec swojego działania, co umożliwia użycie metod łańcuchowo (chaining) podczas tworzenia obiektu (np. builder.append('a').prepend('b').pad('c')). Innymi słowy, brak zwracania wartości przez metodę prowadziłby do sytuacji, w której sekwencja metod nie wpływałaby na ten sam obiekt, ale tworzyłaby nowe, niepowiązane ze sobą obiekty.

Wykorzystanie

W końcowym etapie zastosowano utworzoną klasę. Nowa instancja klasy 'StringBuilder', zainicjalizowana wartością ".", została przypisana do zmiennej o nazwie 'builder'. Następnie, na tej zmiennej wywołano metody w łańcuchu. Każda z tych metod spełniała kolejne wymagania opisane w treści zadania.
